# North Polar Post

Handwritten Santa letter service. Parents purchase a letter package, kids receive personalized handwritten letters from the North Pole via snail mail.

**Dashboard:** https://north-polar-post.onrender.com/admin/dashboard

## Stack
- **Backend:** Node/Express + Prisma + PostgreSQL
- **Payments:** Stripe Checkout + webhooks
- **Handwriting:** Handwrytten API
- **Hosting:** Render
